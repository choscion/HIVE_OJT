기본 폰트사이즈 12px, line-height는 1로 주었다.
IE는 -webkit-box를 지원하지 않기 때문에 height를 따로 잡아주고
넘치는 부분들은 overflow:hidden으로 감춘다
상품명은 두줄로 제한. (scss - mixin 사용)

@mixin elip($line,$lh:null) {
    // 공통
    overflow: hidden;
    text-overflow: ellipsis;
    line-height: $lh+px;
    @if($line > 1){
        display: -webkit-box;
        -webkit-line-clamp: $line;
        -webkit-box-orient: vertical;
        max-height: $line*$lh+px;
    }
    @else{
        white-space: nowrap;
    }
}
